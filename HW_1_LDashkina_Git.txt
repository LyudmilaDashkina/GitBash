JSON

1)Создать внешний репозиторий с названием JSON
Создала новый репозиторий на GitHub с названием JSON

2)Клонировать репозиторий JSON на локальный компьютер
1. скопировала ссылку репозитория
2. git clone https://github.com/LyudmilaDashkina/JSON.git
3. cd JSON/ #Зашла в папку JSON

3)Внутри локального JSON создать файл "new.json"
touch new.json

4)Добавить файл под гит
git add . 
#либо "git add new.json"

5)Закоммитить файл
git commit -m "file json"

6)Отправить файл на внешний репозиторий GitHub
git push

7)Отредактировать содержание файла "new.json" - написать информацию о себе (ФИО, возраст, количество
домашних животных, будущая желаемая зарплата). Все написать в формате JSON.
1. vim new.json
2. i
3. Ввела информацию
4. Esc
5. :wq

8)Отправить изменения на внешний репозиторий GitHub
git add new.json
git commit -m "Modified new.json"
git push

9)Создать файл preferences.json
cat >> preferences.json

10)В файл preferences.json добавить информацию о своих предпочтениях (любимый фильм, любимый сериал, любимая
еда, любимое время года, страна, которую бы хотели посетить) в формате JSON.
1. Ввела информацию
2. ctrl+C

11)Создать файл skills.json и добавить информацию о скиллах, которые будут изучены на курсе в формате JSON
cat >> skills.json
1. Ввела информацию
2. ctrl+C

12)Отправить сразу два файла на внешний репозиторий
git status
git add .
git status
git commit - m "Add files preferences and skills"
git push

13)На ВЕБ интерфейсе создать файл bug_report.json
1. Выбрать в выпадающем меню "Create new file"
2. Назвать файл
3. Нажать "Commit new file"

14)Сделать Commit changes (сохранить) изменения на веб интерфейсе 
1. Open bug_report.json
2. Edit file
3. Commit changes

15)На ВЕБ интерфейсе модифицировать файл bug_report.json, добавить баг репорт в формате JSON
1. Open bug_report.json
2. Edit file, modify

16)Сделать Commit changes (сохранить) изменения на веб интерфейсе
Commit changes

17)Синхронизировать внешний и локальный репозиторий JSON.
git pull
